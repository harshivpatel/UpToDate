FAQs

Q: What is UpToDate?
It's a personalized news aggregator that shows curated news based on your interests.

Q: Can I bookmark articles?
Yes, registered users can bookmark and view saved stories anytime.

Q: Who can access the site?
Anyone can browse news, but only registered users can personalize their experience.

Q: What technologies are used in UpToDate?
The frontend is built using Angular with standalone components and Bootstrap styling. The backend is developed using Node.js and Express.js.

Q: What database is used?
MongoDB is used to store user data, bookmarks, and preferences. Mongoose is used for schema modeling.

Q: How is authentication handled?
We use JWT (JSON Web Tokens) for user authentication and Bcrypt for securely hashing passwords.

Q: Where does the news data come from?
News is fetched using the NewsAPI, allowing category-based and keyword-based news searches.

Q: Is the project deployed?
The current version runs locally, but it's structured to support deployment on platforms like Render, Vercel, or Netlify for frontend and Heroku or Railway for backend.
