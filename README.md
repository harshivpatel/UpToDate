# UpToDate
UpToDate is a dynamic web application that allows users to browse, search, and save news articles from various sources using NewsAPI. 
UpToDate is created using Node.js for backend and Angular for frontend. MongoDB is used as the Database to store the User and User Data.

## Installation Instructions
To set up backend:
cd backend
npm install
node server.js

MONGODB_URI=mongodb://localhost:27017
JWT_SECRET=123456789
PORT:5000
